# body参数

> 参数设置
1. 重力为 0, 对物体进行分类，配置碰撞关系。
2. 球:线性阻尼，角速度阻尼为 1;
3. 球碰撞器的弹性系数为 1;
4. 球杆的能量系数: 18;
5. 球杆最小距离: 20, 最大距离 100;
6. `this.body.applyLinearImpulse`(方向, 世界坐标的质心, true);
7. 设计分辨率: 414x736
