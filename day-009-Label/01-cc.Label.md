# cc.Label

> 知识大纲
1. cc.Label是显示文字的组件
2. cc.Label属性面板
    1. String: 文本显示的内容; 
    2. Horiznotal: 水平对齐的方式: 左 右 居中; 
    3. Vertial: 上, 下, 居中, 字与行的排版; 
    4. Font Size: 字体大小; 
    5. LineHeight: 每行的高度; 
    6. OverFlow:文字排版
        1. None: 没有任何特性; Clamp: 截断; 
        2. Shank:自动缩放到节点大小;  Resize Height: 根据宽度自动折行;
    7. Font: ttf字库文件, 位图字体字库文件 
    8. Font Family: 字体家族,使用系统的哪种字库;
    9. Use System Font: 是否使用系统字体;
3. Label节点所在的锚点修改;
