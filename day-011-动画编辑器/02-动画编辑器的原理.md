# 动画编辑器的原理

> 知识大纲
1. 时间轴
2. 在不同的时刻，调整节点以及孩子节点的不同的属性的值，然后创建出补间动画;
3. 节点调动画的属性:
    * 位置, 缩放, 旋转, 大小, 颜色, 透明度, 锚点, 扭曲, ...;
4. 动画编辑器也可以调节节点的子节点 
5. 动画参数:
    * Simple: 1秒多少帧
    * Speed: 速度,播放速度,越小越慢, 
    * wrapMode:  Normal, Loop, PingPong, Reverse, Loop Reverse, PingPongReverse;
6. 动画
    * 添加动画属性
    * 添加关键帧/删除关键帧,选到关键帧，在属性编辑器上编辑和修改;
    * 编辑补间动画曲线路径;
